# AutoNAS --前端

> 

## 配置要求

`Node >= 6`

## 安装

 - clone这个项目
``` bash
git clone https://github.com/next-NAS/frontend.git
```
 - 进入项目安装依赖

``` bash
npm install
```

## 进行开发

``` bash
# serve with hot reload at localhost:8010
npm run dev
```

## 编译

``` bash
# build for production with minification
npm run build
